# How to run our project

## External libraries
The following libraries are required to run our project:

- `numpy`
- `scikit-learn`
- `h5py`
- `jsonlines`
- `nltk`
- `pandas`
- `vaderSentiment`
- `transformers`
- `torch`

You can install them using one of the followings commands:
```
pip install -r requirements.txt
```
```
conda env create -f environment.yml
```
